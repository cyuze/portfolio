DEVELOPMENT

図書管理アプリ

python, mysql

ユーザーの新規登録とパスワード変更にはメールに送られるワンタイムパスワードを使用しました。ログインすることで登録された図書を借りることができます。管理者権限を持ったアカウントは図書の登録と削除、編集ができます。また、本を借りているアカウントを確認できます。

タイピングゲーム

javascript, html, css

ローカルストレージを用い名前とスコアの記録の確認を可能にしました。モード選択では、単語と文章のどちらかのみの出題を選択できます。通常は10問出題ですが10倍にすることもできます。タイピング終了後のリザルト画面でスコア、レベル、入力時間、入力文字数、ミス入力数、WPM、正確率、苦手キーを表示します。

ポ〇モン風ゲーム(バトル画面)

python

たたかう、バック、ポケモン、にげるのコマンドはすべて実装しています。『たたかう』を選択、二つの技を選べます。『ひっかく』…相手に20のダメージ、『すなかけ』…相手の攻撃命中率を下げる（最大4回まで）。『バック』を選択、『きずぐすり』を選択することで指定した見方の体力を回復できます。『ポケモン』を選択、手持ちの味方のと入れ替えができます。『にげる』…タイトルに戻れます。
